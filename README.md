# Twitter Sentiment Analysis using NLP and Machine Learning

## Project Overview

This project performs sentiment analysis on Twitter data to classify public opinions about different brands and topics using Natural Language Processing (NLP) and Machine Learning techniques.

The project covers:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Text Preprocessing
- TF-IDF Feature Extraction
- Logistic Regression Classification
- Model Evaluation
- Business Insights

---

## Dataset

**Twitter Entity Sentiment Analysis Dataset**

### Dataset Columns

- Tweet ID
- Entity
- Sentiment
- Tweet Text
